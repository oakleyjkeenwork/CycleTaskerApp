# CycleTasker

CycleTasker is a personal offline Android task tracker built with Kotlin, Jetpack Compose, and Material 3.

It focuses on simple recurring individual tasks, custom-length routine cycles, editable today/yesterday completion, calendar history, and local JSON backup/import.

## Principles

- Offline only
- Local JSON storage
- No accounts
- No cloud sync
- No ads
- No internet permission

## Build

Open the project in Android Studio or run:

```powershell
$env:JAVA_HOME='C:\Program Files\Android\Android Studio\jbr'
.\gradlew.bat :app:assembleDebug
```
